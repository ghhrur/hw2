Проект для работы с рецептами, ингредиентами и списком покупок. Реализованы 4 класса: Ingredient, DietaryRecipe, Recipe, ShoppingList.
## Установка
```bash
git clone https://github.com/ghhrur/hw2.git
cd hw2
pip install -r requirements.txt
```
## Использование
Запуск тестов:

```bash
pytest
```
